# 每周 Excel 上传入口

请把各系统导出的最新 `.xlsx` 或 `.xlsm` 文件上传到这个目录。上传后提交，系统会自动重建并发布数据看板。

## 上传规则

1. 各系统可以分别上传不同 Excel 文件，也可以上传包含多个 Sheet 的总底表。
2. 每周可以覆盖同名文件，也可以上传新文件；同一数据来源存在多个版本时，系统自动采用最后上传的版本，避免 YTD 重复累计。
3. 支持的 Sheet：`处方流转订单表`、`BBC商城原版订单表`、`云交易`、`北儿处方`、`珠海双通道`、`慧采`、`地坛HIV`、`互医2.0`。
4. 不要改动原始字段名。未识别的 Sheet 会被跳过，不会影响已发布数据。
5. 上传完成后通常等待 1 到 3 分钟，再刷新公开看板。
